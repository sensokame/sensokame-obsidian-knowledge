# SonarQube
SonarQube is an open-source platform developed by SonarSource for continuous inspection of code quality to perform automatic reviews with static analysis of code to detect bugs, code smells on 17 programming languages

>[SonarQube website](https://www.sonarqube.org/)
>[SonarQube Wikipedia Page](https://en.wikipedia.org/wiki/SonarQube)

## SonarQube rules
SonarQube basically works with a bunch of rules, it scans the quality base and if one of the rules is not respected, a flag will be raised and a call to action is invoked.

>**NOTE:** Users can specify custom rules for sonar qube to search for
>Check [User Guide](https://docs.sonarqube.org/latest/user-guide/rules/) for more information on SonarQube rules
